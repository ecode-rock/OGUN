should i see action in the pgAdmin sql windows?



Yes — open pgAdmin, connect to baseball_db, and run this in the Query Tool:

SELECT pitcher_name, pitch_name, start_speed
FROM pitches
LIMIT 10;

To get there:

Left panel → Servers → PostgreSQL → Databases → baseball_db
Right-click baseball_db → Query Tool
Paste the query above and hit the Execute Query button (▶)
You should also now see the pitches table listed in the object explorer under:

baseball_db → Schemas → public → Tables → pitches

You may need to right-click Tables and hit Refresh if it was already open before the script ran.
